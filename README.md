# Hangman-AIPowered
Hangman game powered with AI and using python,pygame.
It was a group project.
It is a basic hangman game. Firsly, you can choose from where do you want the words. From AI or our database. Previous version of the program, AI part was beta feature but now it works very well. Under of it there are 3 buttons so you can choose difficulty level. Bottom of this screen you can see group members' names.
